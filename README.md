# Port Scanner (Python)

## Overview
A simple TCP port scanner developed in Python for cybersecurity learning and network analysis.

## Features
- Scans common TCP ports
- Supports IP addresses and hostnames
- Displays open ports
- Lightweight and beginner-friendly

## Usage

```bash
python port_scanner.py
```

## Disclaimer
Use only on networks and systems you own or have explicit permission to test.
